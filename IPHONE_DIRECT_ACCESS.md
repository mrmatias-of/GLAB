# Acessar o App G•LAB Direto do iPhone

## Método 1: QR Code (Mais Rápido)

O servidor Expo está rodando agora no servidor Locaweb.

### No seu iPhone:

1. **Abra a câmera**
2. **Aponte para este QR Code:**

```
[QR CODE - Escaneie para abrir o app]

exp://seu-dominio.locaweb.com.br:19000
```

3. **Clique no link que aparecer**
4. **Escolha "Expo Go"**
5. **Pronto! App abre no seu iPhone**

---

## Método 2: Link Direto (Se QR não funcionar)

1. No seu iPhone, instale "Expo Go" (App Store)
2. Abra o app
3. Clique em "Scan QR code"
4. OU manualmente vá para:
   ```
   exp://seu-dominio.locaweb.com.br:19000
   ```

---

## Método 3: Via Web

1. No seu iPhone, abra Safari
2. Vá para: https://seu-dominio.locaweb.com.br:19000
3. Clique em "Open in Expo Go"

---

## O App Funciona com:

- Dashboard: Métricas financeiras
- Ordens: Gerenciamento de OS
- Técnico: Dados do técnico
- Estoque: Gestão de peças
- Perfil: Dados do usuário

Tudo sincronizado em tempo real com https://www.glabcursos.com.br/api

---

## Troubleshooting

**App não conecta:**
- Verifique se o iPhone tem internet
- Verifique se o servidor está rodando

**Expo Go não abre:**
- Instale "Expo Go" na App Store
- Tente novamente

**API não responde:**
- Verifique conexão com www.glabcursos.com.br
