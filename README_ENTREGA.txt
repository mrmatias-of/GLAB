🎯 COMO USAR

1. SETUP INICIAL
   $ git clone https://github.com/mrmatias-of/GLAB.git
   $ cd GLAB
   $ pnpm install

2. CONFIGURAR .env.local
   DATABASE_URL=seu_neon_url
   BETTER_AUTH_SECRET=$(openssl rand -base64 32)
   AI_GATEWAY_API_KEY=sua_chave

3. EXECUTAR
   $ pnpm dev
   Acesso: http://localhost:3000/admin

4. BUILD PRODUÇÃO
   $ pnpm build
   $ pnpm start
